# Level 2 / Phase 06 - Capstone

## Goal

Build, deploy, and document a complete API integration of your own design.

## Requirements

- [ ] Chain at least two third-party APIs.
- [ ] Run on a schedule.
- [ ] Notify you through Discord, Telegram, email, Slack, or another channel.
- [ ] Include tests.
- [ ] Deploy somewhere.
- [ ] Include a README clear enough for someone else to run it.
- [ ] Keep secrets in environment variables.

## Example Projects

| Project | APIs |
|---|---|
| Morning brief | Weather + quote + news headline to Discord |
| Crypto alert | CoinGecko price threshold to Telegram |
| Server monitor | Homelab health checks to Discord |
| GitHub digest | Watched repo issues or stars to Slack |
| DNS monitor | Domain expiry check to Discord |

## Suggested Approach

1. Pick a problem you actually care about.
2. Build the minimum useful version.
3. Add scheduling.
4. Add notification.
5. Add tests.
6. Deploy it.
7. Write the README last.
8. Run it for two weeks before adding extra features.

## Submission

Push to your own repository, deploy it, and share the URL. The final artifact
should be something you can explain in a portfolio or interview.
