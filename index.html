☁️ Save Progress to GitHub Gist

Progress saves to localStorage by default — clearing your cache wipes it. Connect a GitHub token to save to a private Gist instead. Persistent. Cross-device. And using the GitHub API to save your course progress is itself a lesson in how APIs work.

Create a token at github.com/settings/tokens — scope: gist only. Stored in localStorage, sent only to GitHub.
