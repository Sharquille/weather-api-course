Or Run the Course Locally (No Cloudflare Required)

Want to run everything on your own machine without deploying to the web? You can spin up Python's built-in HTTP server to serve the course site and save your progress locally in your browser's localStorage.

  • Service Commandpython3 -m http.server 8080
  • Local URLhttp://localhost:8080
  • Progress SaveBrowser LocalStorage
  • Cloud SyncGitHub Gist (Optional)
# 1. Open your terminal in the repo root:
$ cd weather-api-course

# 2. Run the local Python web server:
$ python3 -m http.server 8080

# 3. Open your browser and navigate to:
http://localhost:8080

Save Progress to GitHub Gist

Progress saves to localStorage by default — clearing your cache wipes it. Connect a GitHub token to save to a private Gist instead. Read the setup guide.

Create a token at github.com/settings/tokens — scope: gist only. Stored in localStorage, sent only to GitHub.
